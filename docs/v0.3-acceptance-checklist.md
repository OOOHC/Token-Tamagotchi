# v0.3 Acceptance Checklist

## Completed

- [x] Bit uses an industrial desktop companion visual direction.
- [x] Default mode keeps the companion compact and desktop-first.
- [x] Expanded mode shows 5-hour token food and 7-day reserve meters.
- [x] Food meters use quota mood colors: happy, relaxed, concerned, panicking, exhausted.
- [x] Details panel uses the same Bit glass HUD language as the food panel.
