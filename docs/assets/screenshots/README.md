# Screenshots

Release screenshots should be placed in this directory.

Recommended captures:

- `bit-compact.png` - default desktop companion with the 5H status HUD.
- `bit-food-panel.png` - expanded 5H token food and 7D reserve meters.
- `bit-details-panel.png` - Details panel with usage and reset information.
- `bit-token-eating.png` - Bit eating token food after a quota drop.

Avoid screenshots that show private project names, prompts, account details, or local file paths.
